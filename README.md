## 🎯 Sobre o Projeto

O **Corrente do Agasalho** é uma iniciativa voltada para organizar o fluxo de doações de roupas, garantindo que os itens cheguem a quem realmente precisa de forma justa e controlada. O sistema gerencia todo o ciclo de vida da doação: desde a entrega em um ponto de coleta até a retirada final pelo beneficiário.

O diferencial do projeto é o uso de um **sistema de pontuação**, que gamifica a solidariedade e estabelece limites saudáveis para as retiradas, priorizando itens de necessidade imediata.

---

## ✨ Funcionalidades Principais

*   **Gestão de Pessoas:** Cadastro de doadores e beneficiários com controle de saldo de pontos limitado.
*   **Locais de Coleta:** Gerenciamento dos pontos físicos onde as doações podem ser entregues.
*   **Registro de Doações:** Entrada de itens no sistema que geram créditos (pontos) para quem doa.
*   **Triagem Inteligente:** As roupas só ficam disponíveis para retirada após passarem por uma curadoria de categoria, tamanho e condição de uso.
*   **Controle de Estoque:** Monitoramento em tempo real do que está disponível para doação.
*   **Gestão de Entregas:** Registro de retiradas de roupas que consomem os pontos dos beneficiários.

---

## ⚙️ Regras de Negócio

Para garantir a equidade do sistema, o projeto segue diretrizes específicas:

1.  **Diferenciação de Pontos:** A pontuação atribuída para quem doa é calculada de forma diferente da pontuação consumida na retirada.
2.  **Itens Prioritários:** Peças essenciais (como cobertores e agasalhos pesados) possuem regras de prioridade em períodos de necessidade.
3.  **Fluxo de Triagem:** Nenhum item entra no estoque disponível sem antes ser validado pela triagem.
4.  **Saldo Limitado:** Cada pessoa possui um limite de pontos para retirada, evitando o acúmulo desnecessário e garantindo a rotatividade do estoque.

---

## 🏛️ Estrutura de Dados

O sistema é estruturado em torno de cinco pilares principais:
*   **Pessoas:** Cadastro central de usuários.
*   **Doações:** Registro de entrada de novos itens.
*   **Itens:** Catálogo de roupas triadas e prontas para uso.
*   **Entregas:** Histórico de saídas e consumo de saldo.
*   **Categorias:** Parametrização de tipos de roupas e pesos de pontuação.
